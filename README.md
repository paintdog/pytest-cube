**pytest cube** is a fun DIY paper cube template featuring some pytest commands, designed for beginners. This project uses the file [Pytest_logo.svg](https://commons.wikimedia.org/wiki/File:Pytest_logo.svg) by [Holger Krekel](https://pypi.org/project/pytest/) licenced [CC BY 2.5](https://creativecommons.org/licenses/by/2.5) from Wikimedia Commons.

## How to Use

1. Download the template file.
2. Print the paper dice template.
3. Cut out and fold along the edges.
4. Glue or tape the dice together.
5. Start rolling and learning pytest!
